# wines

types of wine

```
const wines = require('wines');

console.log(wines.reds());
// ['Cabernet Sauvignon', 'Chianti', 'Malbec', ...]

console.log(wines.whites());
// ['Champagne', 'Chardonnay', 'Pinot Grigio', ...]
```

## Installation

If using `yarn`:
```
yarn add wines
```

Otherwise:
```
npm install wines --save
```
